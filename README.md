# Shenzhen Solitaire

A web-and-mobile-friendly implementation of Shenzhen Solitaire (rules designed by Zachtronics).

Note that drag-and-drop is not supported or needed. Moves are done by selecting the card or stack, then clicking its destination.

Play here: (ad-supported) [https://www.onlinemathlearning.com/shenzhen-solitaire-game.html]

Or here: [https://spheniscine.github.io/solitaire-shenzhen/]
